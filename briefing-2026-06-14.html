AI Market Trends

The June 15 deadline is now a one-day countdown, and a new action item surfaced. Tomorrow, Anthropic moves Claude Agent SDK usage, headless claude -p, Claude Code GitHub Actions, and third-party apps authenticating through the Agent SDK off the subscription usage pool and onto a separate monthly dollar credit ($20 Pro / $100 Max 5x / $200 Max 20x), metered at standard API rates, per user, with no rollover. The same day, claude-sonnet-4-20250514 and claude-opus-4-20250514 retire from the API, after which calls to those IDs return an error. The detail that firmed up today and is easy to miss: the new Agent SDK credit must be explicitly claimed once before June 15 — if it isn't, and usage credits aren't separately enabled, Agent SDK requests can stop until the credit refreshes. The interactive pool — Claude.ai, interactive Claude Code, and Claude Cowork — remains explicitly unaffected.

The migration-target ambiguity is now resolvable at the primary source. For a week, outlets disagreed on whether to migrate retiring Opus 4 workloads to Opus 4.7 or the current flagship. Anthropic's official model-deprecations page settles it: for the June 15 deprecations, the documented default replacements are claude-sonnet-4-5 and claude-opus-4-5, with claude-opus-4-7 suggested for complex or agentic workloads, while Opus 4.8 is the current flagship. The action item is unchanged in spirit but now has an authoritative answer: read the deprecation page, don't trust a single outlet.

A new open-weight entrant landed: Kimi K2.7-Code. Moonshot AI released Kimi K2.7-Code on June 12 — an open-weight, coding-and-agent-focused model, 1 trillion total / ~32B active parameters (mixture-of-experts), weights on Hugging Face under a Modified MIT license permitting commercial use with attribution. Vendor-reported gains over the prior K2.6 include double-digit improvements on the lab's own coding benchmarks and roughly 30% lower thinking-token usage; those figures are self-reported and await independent verification. It is not a frontier launch from a top-three lab, but it is the most substantive new model in the window and matters as open-weight agentic tooling.

The capital backdrop is unchanged: Anthropic's confidential S-1 (filed June 1) still anchors the reported ~$965B valuation / ~$47B annualized revenue picture sourced to trade-press coverage of the filing — not audited.

What this means for foundations and small organizations. The June 15 action is now a same-day-tomorrow checklist: confirm no workflow calls the retiring model IDs; migrate any that do, using the official deprecation page to pick the exact replacement; claim the Agent SDK credit for anyone running scheduled agents; and audit for hard-coded temperature/top_p on the newest Opus. The billing split is a budgeting-and-governance question for clients running scheduled agents — interactive use is unaffected, which stays the reassuring lead for any client note. Kimi K2.7-Code is a clean teaching example of the open-weight-versus-frontier tradeoff: capable and commercially licensable, but its benchmark gains are self-reported.
