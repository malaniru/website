1. AI Market Trends

Anthropic Claude Code pricing experiment — the communication, not the price, is the story. On April 21, Anthropic changed its public pricing page and Claude Code support documentation to show Claude Code as exclusive to the Max plan ($100/month) rather than available in Pro ($20/month). No press release, no changelog entry, no user email. Developer reaction was sharp — Simon Willison, The Register, Wheresyoured.at, and The New Stack all covered it within 24 hours. By April 22, Anthropic's head of growth Amol Avasare posted publicly that the change was "a small test on approximately 2% of new prosumer signups" and "existing Pro and Max subscribers aren't affected," and the pricing page has been partially reverted.

For client-facing posture this is the second material "opaque change" event from Anthropic in two weeks (the first being the effort-reduction parameter narrative from April 13–19). The price point itself is not the concern — the pattern of shipping significant changes without notifying existing users is. For any organization whose governance framework cites "vendor-side unannounced changes" as a risk, today's event is a second data point, not a one-off.
